A Movie Recommendation System built with Flask and Python Libraries. It takes a movie title as an input, and gives out movies that are similar to the inputted movie using OMDb API and cosine similarity of movie title and genres.

## Technologies and Libraries Used:
- **Python**: Programming language used for the backend.
- **Flask**: Web framework used to develop the application.
- **OMDb API**: API used to fetch movie data.
- **TfidVectorizer**: Used to convert movie titles and genres into numerical feature vectors based on their TF-IDF scores.
- **Cosine Similarity**: Used to find similar movies based on title and genre.

## To run the application:
```
python app.py
```
- You can visit the web application after running the above.
